Keyboard shortcuts

  • F8 opens Tabula Rasa from any page.
  • S focuses the search box; Esc clears it.
  • J moves down the list, K moves up.
  • Enter opens the focused tab.
  • Space toggles the focused tab selection.
  • X closes the focused tab item.
  • Ctrl+Enter (or ⌘+Enter) closes all selected tabs.
  • Ctrl/⌘+Click on a tab row closes that tab instantly.
  • A selects or clears all visible tabs.
  • 1, 2, 3 switch between window, recent, and oldest sorting.
  • L toggles between horizontal and vertical layouts.
  • F opens the full-page view (popup only).
  • Q closes the full-page view tab.

Use the Hide pinned toggle to reveal pinned tabs when needed.
