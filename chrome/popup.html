Keyboard shortcuts

While the search box has focus, letter shortcuts type into it. Press Enter once to move focus to the tab list so these hotkeys work.

  • Alt+Shift+Period opens Tabula Rasa from any page.
  • S focuses the search box; Esc clears it.
  • J moves down the list, K moves up.
  • Ctrl+J (or ⌘+J) jumps to the top; Ctrl+K (or ⌘+K) jumps to the bottom.
  • Home jumps to the top; End jumps to the bottom.
  • Enter opens the focused tab.
  • Space toggles the focused tab selection.
  • X closes the focused tab item.
  • Ctrl+Enter (or ⌘+Enter) closes all selected tabs.
  • Ctrl/⌘+Click on a tab row closes that tab instantly.
  • ? opens/closes the hotkeys panel.
  • O opens/closes the options panel.
  • P hides or shows pinned tabs.
  • A selects or clears all visible tabs.
  • 1, 2, 3 switch between window, recent, and oldest sorting.
  • L toggles between horizontal and vertical layouts.
  • F opens the full-page view (popup only).
  • Q closes Tabula Rasa.

Use the Hide pins toggle to reveal pinned tabs when needed.

Advanced Search Features

The search box supports advanced filtering with prefixes and multiple search terms:

  • history: or hist: prefix searches browser history instead of open tabs. Example: history:github or hist:github searches your browsing history for "github".
  • url: prefix restricts search to URLs only. Example: url:github finds tabs with "github" in the URL.
  • title: prefix restricts search to tab titles only. Example: title:docs finds tabs with "docs" in the title.
  • Multiple terms: Space-separated terms use AND logic. Example: test url:git finds tabs with "test" (in title or URL) AND "git" in the URL.
  • Default behavior: Terms without prefixes search both title and URL. Example: mail sync plus finds tabs containing "mail" AND "sync" AND "plus" anywhere.
